Pipeline sketch: S3 upload triggers Lambda for validation, Step Functions orchestrates Glue ETL, and Athena queries parquet outputs for BI. Add partitions and compression early to keep costs down.

# Sample Glue SQL (Athena) partitioned table CREATE EXTERNAL TABLE IF NOT EXISTS analytics.events ( event_time timestamp, user_id string, action string, payload string ) PARTITIONED BY (dt string) STORED AS PARQUET LOCATION 's3://bucket/analytics/events/' TBLPROPERTIES ('parquet.compress'='SNAPPY');

Guide - Web

Shipping a one‑file blog on GitHub Pages

  1. Create a repo and add index.html (this file) at the root.
  2. Commit and push, or upload via the web UI.
  3. Settings → Pages → Branch: main, Folder: /(root) → Save.
  4. Open the URL shown in the Pages section — that’s the blog.

Optional later: add a _config.yml and pick a Jekyll theme; the single‑file site already works fine without a build step.
